# opt-eikana

optキーを英かなキーにする超軽量デーモン

## 動作

- 左option → 英数
- 右option → かな
- option + 他 → そのまま
  - 例: VS Codeのマルチカーソルと干渉しない👍

## インストール

```sh
curl -fsSL https://raw.githubusercontent.com/2754github/opt-eikana/main/install | bash
```

## アンインストール

```sh
curl -fsSL https://raw.githubusercontent.com/2754github/opt-eikana/main/uninstall | bash
```

## 開発に至った経緯

- cmdキーを英かなキーにしていたが、ショートカット誤作動が多く、ストレスだった😢
